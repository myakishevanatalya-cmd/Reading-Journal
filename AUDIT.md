# Аудит приложения

## Исправлено после проверки

- Можно снять ошибочную отметку `прочитано` из дневника книги.
- Прогресс, медали, полка, герои и родительский экран пересчитываются после отмены.
- Квест-вопросы теперь открываются повторно из дневника книги.
- Вопросы идут пошагово: назад / дальше / готово.
- Ответы на вопросы сохраняются и показываются в родительском экране.
- Квест можно открыть повторно из родительской аналитики.
- Service worker обновлен до новой версии кэша.

## Основные сценарии

- Открыть любую книгу с карты или полки.
- Заполнить героя, впечатление, настроение и оценку.
- Сохранить книгу как прочитанную.
- Получить позитивную обратную связь.
- Ответить на вопросы по книге.
- Вернуться к вопросам позже.
- Отменить ошибочно прочитанную книгу.
- Экспортировать и импортировать JSON-резервную копию.
- Сбросить весь прогресс с подтверждением.
- Открыть экран `Прогресс` и увидеть круговую диаграмму, команду героев и этапы маршрута из 32 книг.

## Осознанные ограничения

- Облако Google Sheets подключено через `CLOUD_API_URL`.
- Вопросы универсальные, не по уникальному содержанию каждой книги.
- Полноценная база по произведениям пока не создана.
- Автоматический браузерный прогон в этой среде не выполнялся.
