# Книжная карта лета

Статический SPA-прототип интерактивного читательского дневника по ТЗ.

## Что есть

- карта приключений с рекомендуемым маршрутом из 32 книг;
- прогресс, медали за 5, 10 и 20 прочитанных книг;
- отдельный экран прогресса с круговой диаграммой, командой героев и этапами маршрута;
- полка книг с фильтрами и отдельной вкладкой рекомендуемого маршрута;
- мини-дневник с героем, отзывом, настроением и оценкой;
- отмена ошибочной отметки `прочитано`;
- квест-вопросы после чтения с возможностью вернуться и дописать ответы;
- коллекция героев;
- родительская аналитика и экспорт JSON;
- импорт JSON для восстановления резервной копии;
- сброс прогресса текущего профиля с подтверждением;
- PWA-манифест и service worker.
- визуальное оформление в стиле летнего книжного атласа: баннер, станции-открытки, наклейки, штамп сохранения.

## Как открыть

Откройте `index.html` в браузере. Для разных профилей можно использовать параметр:

```text
index.html?child=child_id_unique&name=Катя
```

Прогресс синхронизируется через Google Таблицу, URL веб-приложения указан в `CLOUD_API_URL` в `script.js`. Локальная копия также хранится в `localStorage` браузера с ключом, зависящим от `child`.

Кнопка сброса находится на родительском экране. Она очищает только данные текущего профиля из ссылки.

Основная иллюстрация шапки лежит в `assets/reading-map-hero.png`.
