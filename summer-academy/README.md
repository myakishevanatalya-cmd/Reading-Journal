# Летняя Академия Кати

Отдельное мини-приложение для летнего закрепления программы 2 класса.

Оно не меняет читательский дневник и хранит данные отдельно:

```text
localStorage key: summer-academy-v1:<child>
```

Пример ссылки:

```text
https://myakishevanatalya-cmd.github.io/Reading-Journal/summer-academy/?child=katya-academy&name=Катя
```

Родительский режим:

```text
https://myakishevanatalya-cmd.github.io/Reading-Journal/summer-academy/?child=katya-academy&name=Катя&parent=1
```

## Что есть

- ежедневная тренировка на 10 заданий;
- отдельный экран тренажеров;
- быстрый счет с учетом времени;
- таблица умножения с выбором числа;
- словарные слова и игровые задания по русскому;
- мягкие шутки и "Веселая перемена";
- темы 2 класса: математика, русский язык, чтение, окружающий мир;
- повтор ошибок;
- карта навыков;
- отчет недели для Кати;
- родительский недельный анализ;
- экспорт и импорт JSON;
- сброс прогресса с подтверждением.

## Принцип

Приложение не ругает за ошибки. Ошибочные задания попадают в повторение и возвращаются в следующих тренировках.
